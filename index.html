  • git clone path-to-repository - клонируем репозиторий (path-to-repository - путь к репозиторию, смотрим в ГИТ)
  • git status - статус текущего локального репозитория
  • git add - добавить новый файл под контроль ГИТА
  • git commit -am "added new file index.html" - фіксуємо зміни локально, в "в лапках коментар"
  • git push - отправить все закомиченные изменения в репозиторий на сервер
  • git pull - забрать свежие обновления с репозитория
